Decisiones con if y else

CONCEPTOS TEÓRICOS

DESICIONES Y COMPARACIONES

Conceptos Teóricos:

  • Sentencias if, else, y elif
    • if: Ejecuta un bloque de código si la condición es verdadera.
    • else: Ejecuta un bloque de código si la condición del if es falsa.
    • elif: Ejecuta un bloque de código si la condición anterior (if o elif) es falsa y la condición actual es verdadera.
  • Comparaciones Simples
    • Igual a (==)
    • No igual a (!=)
    • Mayor que (>)
    • Menor que (<)
    • Mayor o igual que (>=)
    • Menor o igual que (<=)

Programa para determinar si es mayor de edad

1. Entrada de datos:

  • input("Ingresa tu edad: ") muestra un mensaje al usuario para que ingrese su edad.
  • int(...) convierte la entrada del usuario de una cadena de texto (str) a un número entero (int).

2. Condicional if-else:

  • La sentencia if verifica si la edad ingresada es mayor o igual a 18.
  • Si la condición edad >= 18 es verdadera, se ejecuta print("Eres mayor de edad.").
  • Si la condición es falsa, se ejecuta el bloque else, que imprime print("Eres menor de edad.").

Funcionamiento

  • El usuario ingresa su edad.
  • El programa determina si la persona es mayor o menor de edad y muestra el mensaje correspondiente.

Programa para determinar la calificación basada en la nota

1. Asignación de la variable nota:

  • Aquí se asigna directamente el valor 85 a la variable nota.

2. Condicional if-elif-else:

  • La sentencia if verifica si la nota es mayor o igual a 90.
  • Si la condición nota >= 90 es verdadera, se ejecuta print("Tienes una A.").
  • Si no, se evalúa la siguiente condición elif, que verifica si la nota es mayor o igual a 80.
  • Si nota >= 80 es verdadera, se ejecuta print("Tienes una B.").
  • Si ninguna de las condiciones anteriores es verdadera, se evalúa la siguiente condición elif, que verifica si la nota es mayor o igual a 70.
  • Si nota >= 70 es verdadera, se ejecuta print("Tienes una C."). Si ninguna de las condiciones anteriores es verdadera, se ejecuta el bloque else, que imprime print("Necesitas mejorar.").

Funcionamiento

  • El valor de la nota se evalúa para determinar la calificación.
  • Dependiendo del valor de la nota, se imprime un mensaje que indica la calificación (A, B, C) o la necesidad de mejorar.

Resumen

  • if: Ejecuta un bloque de código si la condición es verdadera.
  • elif: Proporciona condiciones adicionales si la condición if original es falsa.
  • else: Captura todos los demás casos cuando ninguna de las condiciones anteriores es verdadera.

Estas estructuras se utilizan en conjunto para permitir que el programa tome decisiones y ejecute diferentes bloques de código basados en diferentes condiciones.

ACTIVIDADES

PROGRAMAS QUE TOMAN DECISIONES BASADAS EN LAS ENTRADAS DEL USUARIO

Crear un programa que tome la edad del usuario y determine si es mayor de edad.

  1. Entrada de datos

    edad = int(input("Ingresa tu edad: "))
    • input("Ingresa tu edad: ") muestra un mensaje al usuario para que ingrese su edad.
    • int(...) convierte la entrada del usuario de una cadena de texto (str) a un número entero (int).

  2. Condicional if-else:

    if edad >= 18: print("Eres mayor de edad.") else: print("Eres menor de edad.")
    • La sentencia if verifica si la edad ingresada es mayor o igual a 18.
    • Si la condición edad >= 18 es verdadera, se ejecuta print("Eres mayor de edad.").
    • Si la condición es falsa, se ejecuta el bloque else, que imprime print("Eres menor de edad.").

3. Funcionamiento

  • El usuario ingresa su edad cuando se le solicita.
  • El programa convierte esta entrada a un número entero.
  • Luego, el programa evalúa si la edad es 18 o mayor.
  • Dependiendo del resultado de esta evaluación, el programa imprime un mensaje indicando si el usuario es mayor o menor de edad.

ADIVINAR SI UN NÚMERO ES MAYOR, MENOR O IGUAL A 10

Crear un programa que tome un número del usuario y adivine si es mayor, menor o igual a 10.

  1. Entrada de datos:

    numero = int(input("Ingresa un número: "))
    • input("Ingresa un número: ") muestra un mensaje al usuario para que ingrese un número.
    • int(...) convierte la entrada del usuario de una cadena de texto (str) a un número entero (int).

  2. Condicional if-elif-else:

    • La sentencia if verifica si el número ingresado es mayor que 10.
    • Si la condición numero > 10 es verdadera, se ejecuta print("El número es mayor que 10.").
    • Si no, se evalúa la condición elif, que verifica si el número es menor que 10.
    • Si numero < 10 es verdadero, se ejecuta print("El número es menor que 10.").
    • Si ninguna de las condiciones anteriores es verdadera (es decir, si el número no es ni mayor ni menor que 10), se ejecuta el bloque else, que imprime print("El número es igual a 10.")

   3. Funcionamiento

  • El usuario ingresa un número cuando se le solicita.
  • El programa convierte esta entrada a un número entero.
  • Luego, el programa evalúa tres condiciones:
    1. Si el número es mayor que 10.
    2. Si el número es menor que 10.
    3. Si el número es igual a 10.
  • Dependiendo del resultado de estas evaluaciones, el programa imprime un mensaje correspondiente.

ASIGNACIÓN DE TAREAS Y CIERRE

  • Tarea:
    • Escribir un programa que diga si un número es par o impar.
  • Cierre:
    • Resumir, en el cuaderno y a mano,  los puntos clave de la sesión

Objetivo

Crea un programa en Python que determine si un número ingresado por el usuario es par o impar e imprima el resultado.

Instrucciones

  1. Solicita un número al usuario y convierte la entrada a un número entero:

    • Usa input para solicitar un número al usuario.
    • Convierte la entrada del usuario a un número entero usando int.

  2. Determina si el número es par o impar:

    • Usa una estructura if-else para determinar si el número es par o impar.
    • Imprime el resultado.

Ejemplo de Código

Explicación Detallada

Paso 1: Solicitar un número al usuario y convertirlo a un número entero

  • input("Ingresa un número: ") muestra un mensaje al usuario para que ingrese un número.
  • int(...) convierte la entrada del usuario (que es una cadena de texto) en un número entero y lo almacena en la variable numero.

Paso 2: Determinar si el número es par o impar usando una estructura if-else

  • if numero % 2 == 0: verifica si el número ingresado es par calculando el residuo de la división del número entre 2.
    • La expresión numero % 2 calcula el residuo de dividir numero entre 2.
    • Si el residuo es 0, entonces numero es par, porque es divisible por 2 sin ningún residuo.
  • Si la condición numero % 2 == 0 es verdadera, se ejecuta print("El número es par.").
  • Si la condición es falsa, se ejecuta el bloque else, que imprime print("El número es impar.").

Funcionamiento

  1. El usuario ingresa un número:

    • El usuario ve el mensaje Ingresa un número: y escribe un número, por ejemplo, 4.
    • El programa convierte esta entrada a un entero y la almacena en la variable numero.

  2. El programa evalúa si el número es par o impar:

    • La expresión numero % 2 == 0 verifica si el residuo de la división del número entre 2 es 0.
    • Si el número es 4, entonces 4 % 2 es 0, lo que significa que 4 es par, y el programa imprimirá "El número es par."
    • Si el número es 5, entonces 5 % 2 es 1, lo que significa que 5 es impar, y el programa imprimirá "El número es impar."

Resultado Esperado

Cuando ejecutes el programa, deberías ver algo similar a esto en la pantalla:

  • Ingresa un número: 4 El número es par.
  • Ingresa un número: 5 El número es impar.

Consejos

  • Asegúrate de ingresar un número entero cuando el programa te lo solicite.
  • Usa la función print para mostrar el mensaje de manera clara y legible.
  • Revisa tu código para asegurarte de que no haya errores de sintaxis y que la variable esté correctamente definida.
