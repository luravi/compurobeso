Valores, Tipos y Variable

CONCEPTOS TEÓRICOS

TIPOS DE DATOS : NÚMEROS Y CADENAS

  • Números: Pueden ser enteros (int) o decimales (float).

  • Cadenas: Texto encerrado entre comillas (str).

VARIABLES

  • Una variable es un espacio en la memoria que se utiliza para almacenar un valor que puede cambiar durante la ejecución del programa.

Cómo nombrar variables

  • Las variables deben tener nombres descriptivos.
  • Deben comenzar con una letra o un guion bajo _.
  • Pueden contener letras, números y guion bajo _.
  • No pueden usar palabras reservadas de Python como print, if, else, etc.

        Este código define cuatro variables y les asigna diferentes tipos de valores.

        1. Variable edad:

          • edad = 12
            • Se crea una variable llamada edad y se le asigna el valor entero 12.
            • El tipo de dato de esta variable es un entero (int), que representa un número sin decimales.

        2. Variable altura:

          • altura = 1.75 
            • Se crea una variable llamada altura y se le asigna el valor decimal 1.75.
            • El tipo de dato de esta variable es un número de punto flotante (float), que representa un número con decimales.

        3. Variable nombre:

          • nombre = "Ana" 
            • Se crea una variable llamada nombre y se le asigna la cadena de texto "Ana".
            • El tipo de dato de esta variable es una cadena de caracteres (str), que representa texto.

        4. Variable es_estudiante

          • es_estudiante = True 
            • Se crea una variable llamada es_estudiante y se le asigna el valor booleano True.
            • El tipo de dato de esta variable es un booleano (bool), que representa un valor de verdad (True para verdadero y False para falso).

        Resumen:

        • edad: Es una variable entera que almacena la edad, en este caso 12.
        • altura: Es una variable de punto flotante que almacena la altura en metros, en este caso 1.75.
        • nombre: Es una variable de cadena que almacena el nombre, en este caso "Ana".
        • es_estudiante: Es una variable booleana que indica si es estudiante, en este caso True (verdadero).

        Estas variables pueden ser usadas más adelante en el programa para realizar operaciones, mostrar información, o tomar decisiones basadas en sus valores.

        CREAR UN JUEGO USANDO VARIABLES

        • Crea un programa que lleve la cuenta de los puntos de dos jugadores en un juego.

              Lectura facilitada

              Explicación del código:

              1. Asignación de valores a variables:

                • jugador1 = "Ana":

                  • Se crea una variable llamada jugador1 y se le asigna la cadena de texto "Ana".
                  • Esta variable almacena el nombre del primer jugador.

                • jugador2 = "Luis":

                  • Se crea una variable llamada jugador2 y se le asigna la cadena de texto "Luis".
                  • Esta variable almacena el nombre del segundo jugador.

                • puntos_jugador1 = 5:

                  • Se crea una variable llamada puntos_jugador1 y se le asigna el valor entero 5.
                  • Esta variable almacena la cantidad de puntos que tiene el primer jugador.

                • puntos_jugador2 = 3:

                  • Se crea una variable llamada puntos_jugador2 y se le asigna el valor entero 3.
                  • Esta variable almacena la cantidad de puntos que tiene el segundo jugador.

              2. Uso de la función print para mostrar información:

                • print("Puntuación del juego:"):

                  • La función print se utiliza para mostrar información en la pantalla.
                  • En este caso, imprime la cadena de texto "Puntuación del juego:", que sirve como título o encabezado para la información que sigue.

                • print(jugador1, "tiene", puntos_jugador1, "puntos."):

                  • La función print imprime varios elementos en una sola línea, separados por espacios.
                  • jugador1 es la variable que contiene el nombre "Ana".
                  • "tiene" es una cadena literal que se imprimirá tal cual.
                  • puntos_jugador1 es la variable que contiene el valor 5.
                  • "puntos." es otra cadena literal que se imprimirá tal cual.
                  • El resultado de esta línea será: Ana tiene 5 puntos.

                • print(jugador2, "tiene", puntos_jugador2, "puntos."):

                  • Similar a la línea anterior, esta línea imprime varios elementos en una sola línea.
                  • jugador2 es la variable que contiene el nombre "Luis".
                  • "tiene" es una cadena literal que se imprimirá tal cual.
                  • puntos_jugador2 es la variable que contiene el valor 3.
                  • "puntos." es otra cadena literal que se imprimirá tal cual.
                  • El resultado de esta línea será: Luis tiene 3 puntos.

              Resumen del funcionamiento:

              El código define dos variables para almacenar los nombres de los jugadores (jugador1 y jugador2) y dos variables para almacenar sus respectivas puntuaciones (puntos_jugador1 y puntos_jugador2). Luego, utiliza la función print para mostrar la puntuación del juego en un formato legible. Primero, imprime un encabezado ("Puntuación del juego:") y luego, imprime las puntuaciones de cada jugador en líneas separadas, combinando texto literal y valores de las variables para formar frases completas.

              Apoyo visual

              Explicación del código:

              1. Asignación de valores a variables:

                • nombre = "Juan"

                  • Se crea una variable llamada nombre y se le asigna la cadena de texto "Juan".
                  • Esta variable almacena un nombre en forma de texto (cadena de caracteres o string).

                • edad = 12

                  • Se crea una variable llamada edad y se le asigna el valor entero 12.
                  • Esta variable almacena la edad como un número entero.

                • altura = 1.60

                  • Se crea una variable llamada altura y se le asigna el valor decimal 1.60.
                  • Esta variable almacena la altura en metros como un número de punto flotante (float).
                  • El comentario # en metros indica que la altura se está midiendo en metros.

              2. Uso de la función print para mostrar información:

                • print("Nombre:", nombre)

                  • La función print se utiliza para mostrar información en la pantalla.
                  • "Nombre:" es una cadena literal que se imprimirá tal cual.
                  • nombre es la variable que contiene el valor "Juan", y su valor se imprimirá después de la cadena literal.
                  • El resultado será: Nombre: Juan.

                • print("Edad:", edad, "años")

                  • "Edad:" es una cadena literal.
                  • edad es la variable que contiene el valor 12.
                  • "años" es otra cadena literal que se imprimirá después del valor de edad.
                  • Todos estos elementos se imprimirán en una sola línea, separados por espacios.
                  • El resultado será: Edad: 12 años.

                • print("Altura:", altura, "metros")

                  • "Altura:" es una cadena literal.
                  • altura es la variable que contiene el valor 1.60.
                  • "metros" es otra cadena literal que se imprimirá después del valor de altura.
                  • Todos estos elementos se imprimirán en una sola línea, separados por espacios.
                  • El resultado será: Altura: 1.60 metros.

              Resumen del funcionamiento:

              El código define tres variables (nombre, edad y altura) y les asigna valores específicos. Luego, utiliza la función print para mostrar estos valores en la pantalla con mensajes descriptivos. Cada llamada a print combina texto literal con los valores de las variables para generar una salida legible y bien formada.

              Explicación del código:

              2. Uso de la función print para mostrar información:

                • print("Tengo", edad, "años."):

                    • "años.": Es otra cadena literal que se imprimirá tal cual.
                  • La función print combina estos elementos separados por comas y los imprime en una sola línea con espacios entre ellos, resultando en el mensaje: Tengo 25 años.
