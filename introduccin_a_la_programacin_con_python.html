Introducción a la programación con Python

CONCEPTOS TEÓRICOS

¿QUÉ ES LA PROGRAMACIÓN Y CÓMO NOS AYUDA EN LA VIDA DIARIA?

Programación es el proceso de decirle a una computadora qué hacer a través de instrucciones específicas. Estas instrucciones se escriben en lenguajes de programación como Python, que es como el idioma que entiende la computadora.

Cómo nos ayuda en la vida diaria:

a)      Automatización de tareas: La programación permite que las computadoras realicen tareas repetitivas automáticamente, como enviar mensajes, hacer copias de seguridad de archivos y organizar datos.

b)     Desarrollo de aplicaciones: Aplicaciones que usamos todos los días, como juegos en el teléfono, aplicaciones de redes sociales y programas de dibujo, son creadas gracias a la programación.

c)      Interacción con la tecnología: Muchos dispositivos, como teléfonos inteligentes, tabletas y televisores inteligentes, funcionan gracias a programas que les dicen qué hacer.

d)     Acceso a información: Gracias a la programación, podemos buscar información en Google, ver videos en YouTube y comunicarnos con amigos y familiares en aplicaciones como WhatsApp.

e)      Solución de problemas: Los programas ayudan a resolver problemas en diferentes áreas, como la medicina (diagnosticar enfermedades), la educación (enseñar de forma interactiva) y el transporte (navegación por GPS).

EJEMPLOS DE PROGRAMAS Y APLICACIONES QUE USAN PYTHON

A.     Web Development:

  • Django y Flask: Son herramientas que ayudan a crear sitios web. Por ejemplo, Instagram y Pinterest usan Django.

B.     Ciencia de Datos y Machine Learning:

  • Pandas, NumPy y SciPy: Son bibliotecas que ayudan a los científicos a analizar datos, como hacer cálculos y gráficos.
  • TensorFlow y PyTorch: Son herramientas que permiten a las computadoras aprender de los datos. Google usa TensorFlow para cosas como la búsqueda inteligente y las recomendaciones.

C.    Automatización de Tareas:

  • Selenium: Se usa para probar automáticamente si los sitios web funcionan correctamente.
  • Scripts de Python: Son pequeños programas que hacen tareas específicas, como renombrar muchos archivos a la vez o convertir archivos de un formato a otro.

D.    Desarrollo de Juegos:

  • Pygame: Es una herramienta que permite crear juegos en 2D, como un juego de carreras o un juego de plataformas simple.

E.      Aplicaciones de Escritorio:

  • Tkinter: Es una herramienta estándar de Python para crear programas con ventanas, botones y menús.
  • PyQt y Kivy: Son herramientas avanzadas para crear aplicaciones de escritorio con interfaces bonitas y útiles.

F.      Redes y Seguridad:

  • Paramiko: Se usa para trabajar de manera segura con otros equipos.
  • Scapy: Ayuda a analizar y modificar datos que viajan por las redes.

G.    Educación:

  • Jupyter Notebooks: Son herramientas interactivas que ayudan a aprender programación y a analizar datos de manera fácil y visual.

  • Tarea:
    • Escribir un programa que imprima un mensaje de bienvenida con tu nombre.
  • Cierre:
    • Resumir, en el cuaderno y a mano,  los puntos clave de la sesión

Explicación del código:

  1. Asignación de una cadena a una variable:

    • nombre = "Ana": Esta línea crea una variable llamada nombre y le asigna la cadena de texto "Ana". Una variable es como una caja en la que podemos guardar datos para usarlos más adelante en el programa.

  2. Concatenación de cadenas y uso de la función print:

    • print("¡Hola, " + nombre + "! Bienvenido al curso de Python."):
      • La función print se utiliza para mostrar texto en la pantalla.
      • Dentro de print, estamos concatenando (uniendo) varias cadenas de texto:
        • "¡Hola, ": Esta es una cadena literal, que siempre será la misma.
        • nombre: Esta es la variable que contiene el texto "Ana". Al incluir esta variable en la concatenación, estamos insertando su valor en medio de las cadenas literales.
        • "! Bienvenido al curso de Python.": Esta es otra cadena literal.
      • El símbolo + se usa para concatenar (unir) las cadenas de texto y la variable nombre.

Resultado del código:

Cuando se ejecuta este código, la función print mostrará en la pantalla el siguiente mensaje:

El valor de la variable nombre se inserta dentro del mensaje, haciendo que el saludo sea personalizado. En este caso, como nombre tiene el valor "Ana", el mensaje resultante incluye este nombre. Si cambiaras el valor de nombre, el mensaje se actualizaría automáticamente para incluir el nuevo nombre.
