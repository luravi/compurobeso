Programación con el Robot Maqueen

1. Primeros pasos con Maqueen

Planteamiento del problema

El objetivo de este programa es controlar un robot Maqueen utilizando la micro para que realice una secuencia específica de movimientos. El robot debe avanzar hacia adelante a máxima velocidad durante 1 segundo, detenerse por medio segundo, luego retroceder a máxima velocidad durante 1 segundo, y finalmente detenerse nuevamente. Esta secuencia de movimientos puede utilizarse para realizar maniobras simples, como simular un movimiento de ida y vuelta.

Solución con MICROPYTHON

Código en Python para micro:bit:

Explicación del Código:

  1. Mover hacia adelante a velocidad máxima

    maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CW, 255)
    • Propósito: Esta línea hace que el robot avance hacia adelante a su velocidad máxima.
    • Explicación:
      • maqueen.motor_run: Esta función se utiliza para activar los motores del robot.
      • maqueen.Motors.ALL: Este parámetro indica que ambos motores, izquierdo (M1) y derecho (M2), deben activarse.
      • maqueen.Dir.CW: Este parámetro indica la dirección en la que deben girar los motores, CW significa "Clockwise" (en sentido horario), lo que hace que el robot avance.
      • 255: Este es el valor de la velocidad, donde 255 es la velocidad máxima.

  2. Espera 1 segundo

    basic.pause(1000)
    • Propósito: Pausa la ejecución del programa durante 1000 milisegundos (1 segundo).
    • Explicación:
      • basic.pause(1000): Esta función detiene temporalmente el programa, permitiendo que el robot avance durante 1 segundo antes de realizar la siguiente acción.

  3. Detener los motores

    maqueen.motor_stop(maqueen.Motors.ALL)
    • Propósito: Esta línea detiene ambos motores del robot.
    • Explicación:
      • maqueen.motor_stop: Esta función se usa para detener los motores.
      • maqueen.Motors.ALL: Detiene ambos motores del robot, dejando el robot inmóvil.

  4. Espera medio segundo

    basic.pause(500)
    • Propósito: Pausa la ejecución del programa durante 500 milisegundos (medio segundo).
    • Explicación:
      • basic.pause(500): Esta función detiene temporalmente el programa durante medio segundo, permitiendo que el robot permanezca quieto antes de ejecutar el siguiente movimiento.

  5. Mover hacia atrás a velocidad máxima

    maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CCW, 255)
    • Propósito: Hace que el robot retroceda a su velocidad máxima.
    • Explicación:
      • maqueen.motor_run: Activa los motores del robot.
      • maqueen.Motors.ALL: Este parámetro indica que ambos motores deben activarse.
      • maqueen.Dir.CCW: Este parámetro indica la dirección en la que deben girar los motores, CCW significa "Counterclockwise" (en sentido antihorario), lo que hace que el robot retroceda.
      • 255: Es la velocidad máxima a la que el robot se moverá hacia atrás.

  6. Espera 1 segundo

    basic.pause(1000)
    • Propósito: Pausa la ejecución del programa durante 1 segundo.
    • Explicación:
      • basic.pause(1000): Esta función detiene temporalmente el programa, permitiendo que el robot retroceda durante 1 segundo antes de detenerse.

  7. Detener los motores

    maqueen.motor_stop(maqueen.Motors.ALL)
    • Propósito: Detiene ambos motores del robot.
    • Explicación:
      • maqueen.motor_stop: Detiene los motores del robot.
      • maqueen.Motors.ALL: Detiene ambos motores, dejando el robot inmóvil después de completar el movimiento hacia atrás.

    3. Trompos con Maqueen

    Planteamiento del problema

    El objetivo de este programa es controlar un robot Maqueen utilizando la micro para que realice una serie de movimientos secuenciales. El robot debe realizar un ciclo de movimiento donde el motor izquierdo avance mientras el motor derecho retrocede, luego detenerse, y después realizar un movimiento inverso en el que el motor izquierdo retroceda y el motor derecho avance. Este ciclo de movimientos se repetirá de manera continua.

    Solución con  MICROPYTHON

    Código en Python para micro:bit:

    Explicación detallada del código en Python

    1. Definición de la función on_forever:

      def on_forever():
      • Propósito: Esta línea define una función llamada on_forever que contiene las instrucciones que el robot Maqueen ejecutará de manera repetida. La función engloba todas las acciones que se quieren realizar en cada ciclo.

    2. Primera secuencia de movimiento:

      • Motor izquierdo (M1) avanza en sentido horario

        maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, 50)
        • Propósito: Activa el motor izquierdo (M1) para avanzar en sentido horario (CW) a una velocidad de 50.
        • Explicación: El motor izquierdo gira en sentido horario, lo que en combinación con el movimiento del motor derecho hace que el robot gire hacia la derecha.

      • Motor derecho (M2) retrocede en sentido antihorario

        maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CCW, 50)
        • Propósito: Activa el motor derecho (M2) para retroceder en sentido antihorario (CCW) a una velocidad de 50.
        • Explicación: El motor derecho gira en sentido contrario, lo que contribuye al giro del robot hacia la derecha.

      • Pausa para permitir el movimiento

        basic.pause(1000)
        • Propósito: Pausa la ejecución del programa durante 1 segundo, permitiendo que el robot realice el giro completo hacia la derecha.
        • Explicación: Durante este tiempo, el robot estará realizando el movimiento de giro.

    3. Primera parada de los motores

      • Detener ambos motores:

        maqueen.motor_stop(maqueen.Motors.ALL)
        • Propósito: Detiene ambos motores del robot para finalizar el giro.
        • Explicación: El robot se detiene por completo al finalizar la primera secuencia de movimiento.

      • Pausa para mantener el robot detenido

        basic.pause(500)
        • Propósito: Pausa la ejecución del programa durante 0.5 segundos para que el robot permanezca detenido antes de comenzar la siguiente secuencia.
        • Explicación: Este breve período de pausa asegura que haya una separación clara entre las dos secuencias de movimiento.

    4. Segunda secuencia de movimiento

      • Motor izquierdo (M1) retrocede en sentido antihorario

        maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CCW, 50)
        • Propósito: Activa el motor izquierdo (M1) para retroceder en sentido antihorario (CCW) a una velocidad de 50.
        • Explicación: Ahora el motor izquierdo gira en sentido contrario, lo que en combinación con el motor derecho hace que el robot gire hacia la izquierda.

      • Motor derecho (M2) avanza en sentido horario

        maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, 50)
        • Propósito: Activa el motor derecho (M2) para avanzar en sentido horario (CW) a una velocidad de 50.
        • Explicación: El motor derecho gira en sentido horario, lo que contribuye al giro del robot hacia la izquierda.

      • Pausa para permitir el movimiento

        basic.pause(1000)
        • Propósito: Pausa la ejecución del programa durante 1 segundo, permitiendo que el robot realice el giro completo hacia la izquierda.
        • Explicación: Durante este tiempo, el robot estará realizando el movimiento de giro en la dirección opuesta a la primera secuencia.

    5. Segunda parada de los motores

      • Detener ambos motores

        maqueen.motor_stop(maqueen.Motors.ALL)
        • Propósito: Detiene ambos motores del robot para finalizar el segundo giro.
        • Explicación: El robot se detiene por completo después de la segunda secuencia de movimiento.

      • Pausa para mantener el robot detenido

        basic.pause(500)
        • Propósito: Pausa la ejecución del programa durante 0.5 segundos para que el robot permanezca detenido antes de repetir el ciclo.
        • Explicación: Esta pausa asegura que haya una separación clara antes de que el ciclo de movimientos se repita.

    6. Ejecución continua

      basic.forever(on_forever)
      • Propósito: Esta línea hace que la función on_forever se ejecute repetidamente en un bucle infinito.
      • Explicación: El ciclo de movimientos definidos en la función on_forever se repetirá indefinidamente, haciendo que el robot continúe girando hacia la derecha e izquierda de manera alternada.

      4. Evita obstáculos

      Planteamiento del problema

      El objetivo es diseñar un programa en MicroPython para la micro que controle un robot Maqueen. El robot debe avanzar hacia adelante, detectar obstáculos usando su sensor ultrasónico, y evitar estos obstáculos. Para ello, si se detecta un obstáculo a menos de 10 cm, el robot debe retroceder y girar antes de continuar avanzando. Este ciclo se repetirá indefinidamente para asegurar que el robot pueda navegar sin colisiones.

      Solución con MICROPYTHON

      Instalar la Extensión Maqueen

      1. Abrir MakeCode para micro
        • Abre el editor MakeCode en tu navegador web.
      2. Añadir la Extensión Maqueen:
        • Haz clic en Avanzado (en la parte inferior del menú de bloques).
        • Luego selecciona Extensiones.
        • En el cuadro de búsqueda, escribe "Maqueen" y selecciona la extensión "Maqueen" de la lista de resultados.
        • Esto añadirá los bloques de Maqueen a tu área de trabajo en MakeCode.

      Código en Python para micro:bit

      Explicación del Código en Python

      • Definición de la función on_forever:

        def on_forever():
        • Propósito: Esta línea define una función llamada on_forever que se ejecutará de manera repetida e indefinida. En ella se especifica el comportamiento del robot cuando se encuentra con un obstáculo.

      • Medir la distancia con el sensor ultrasónico:

        if maqueen.ultrasonic(PingUnit.CENTIMETERS) < 10:
        • Propósito: Esta línea utiliza el sensor ultrasónico del robot para medir la distancia a un objeto frente a él. Si la distancia medida es menor a 10 centímetros, se considera que hay un obstáculo cerca.
        • Explicación: maqueen.ultrasonic(PingUnit.CENTIMETERS) mide la distancia al objeto en centímetros, y el código verifica si esa distancia es menor que 10.

      • Si hay un obstáculo cerca, retrocede:

        • Motor izquierdo y derecho retroceden en sentido antihorario:

          maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CCW, 50) maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CCW, 50)
          • Propósito: Activa ambos motores (M1 y M2) para que retrocedan en sentido antihorario (CCW) a una velocidad de 50.
          • Explicación: Esto hace que el robot retroceda para alejarse del obstáculo.

        • Pausa para permitir el movimiento de retroceso:

          basic.pause(1000)
          • Propósito: Pausa la ejecución del programa durante 1 segundo, permitiendo que el robot retroceda durante ese tiempo.
          • Explicación: El robot se moverá hacia atrás durante 1 segundo.

        • Detener ambos motores:

          maqueen.motor_stop(maqueen.Motors.ALL)
          • Propósito: Detiene ambos motores del robot, dejando de moverse.
          • Explicación: Esto detiene al robot después de haber retrocedido.

        • Pausa breve

          basic.pause(500)
          • Propósito: Pausa la ejecución del programa durante 0.5 segundos para asegurar que el robot esté detenido antes de realizar la siguiente maniobra.
          • Explicación: Esta breve pausa garantiza que haya una separación clara entre las acciones.

      • Girar hacia la derecha para esquivar el obstáculo:

        • Función: Este bloque se ejecuta una sola vez al inicio del programa, cuando la micro se enciende o reinicia. En este caso, se utiliza para inicializar la variable derecha.
          • Configuración y Subbloque 
          • Lugar  Variables , crea tu variable  con el nombre "derecha". 
          • En el bloque "al iniciar, " desde el bloque de variables arrastra " fijar derecha a" y en el "0" que aparece arrastrar desde el bloque matemáticas escoge al azar un valor verdadero o falso. Este bloque se usa para inicializar la variable derecha con un valor booleano (true o false) de manera aleatoria.

      2. Bloque "para siempre"

        • Ubicación: Básico > para siempre.
        • Función: Este bloque crea un bucle infinito que repite continuamente las acciones en su interior, permitiendo que el robot revise continuamente el entorno y ajuste su movimiento.

      3. Bloque "si (leer ultrasonido < 20 cm)"

        • Ubicación: Lógica > si...entonces y Maqueen > Ultrasonido > Leer ultrasonido.
        • Función: Verifica si la distancia medida por el sensor ultrasónico es menor a 20 cm.
        • Configuración:
          • Leer Ultrasonido: En Maqueen > Ultrasonido, selecciona "Leer ultrasonido en centímetros". Configura la comparación para verificar si la distancia es menor a 20.

      4. Bloque "Motor: ambos, sentido avanzar, velocidad 200"

        • Ubicación: Maqueen > Motor > Motor run.
        • Función: Activa ambos motores (M1 y M2) para que el robot avance en sentido horario (CW) a una velocidad de 200.
        • Configuración:
          • Motor: Selecciona ambos para activar tanto el motor izquierdo como el derecho.
          • Sentido: Selecciona avanzar para que ambos motores giren en sentido horario.
          • Velocidad: Ajusta a 200 para que el robot avance a una velocidad moderada.

      5. Bloque "si no"

        • Ubicación: Lógica > si no.
        • Función: Este bloque permite especificar lo que debe hacer el robot cuando no se detecta ningún objeto cercano.

      6. Subbloque "si (derecha == True)"

        • Ubicación: Lógica > si...entonces.
        • Función: Verifica si la variable derecha es True, lo que significa que el robot debe girar hacia la derecha.

      7. Subbloque "Motor: M1, sentido avanzar, velocidad 200" y "Motor: M2, sentido detener, velocidad 0"

        • Ubicación: Maqueen > Motor > Motor run y Motor stop.
        • Función: Hace que el motor izquierdo avance mientras que el motor derecho se detiene, haciendo que el robot gire hacia la derecha.
        • Configuración:
          • Motor izquierdo:
            • Motor: Selecciona M1 para activar solo el motor izquierdo.
            • Sentido: Selecciona avanzar para que el motor izquierdo gire en sentido horario.
            • Velocidad: Ajusta a 200 para que el motor avance a una velocidad moderada.
          • Motor derecho:
            • Motor: Selecciona M2 para activar solo el motor derecho.
            • Sentido: Selecciona detener para que el motor derecho no gire.
            • Velocidad: Ajusta a 0 para detener el motor derecho.

      8. Subbloque "si no"

        • Ubicación: Lógica > si no.
        • Función: Si derecha es False, se ejecutan las instrucciones en este subbloque, que hacen que el robot gire hacia la izquierda.

      9. Subbloque "Motor: M1, sentido detener, velocidad 0" y "Motor: M2, sentido avanzar, velocidad 200"

        • Ubicación: Maqueen > Motor > Motor run y Motor stop.
        • Función: Hace que el motor derecho avance mientras que el motor izquierdo se detiene, haciendo que el robot gire hacia la izquierda.
        • Configuración:
          • Motor izquierdo:
            • Motor: Selecciona M1 para activar solo el motor izquierdo.
            • Sentido: Selecciona detener para que el motor izquierdo no gire.
            • Velocidad: Ajusta a 0 para detener el motor izquierdo.
          • Motor derecho:
            • Motor: Selecciona M2 para activar solo el motor derecho.
            • Sentido: Selecciona avanzar para que el motor derecho gire en sentido horario.
            • Velocidad: Ajusta a 200 para que el motor avance a una velocidad moderada.
