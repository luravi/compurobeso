Funciones II y Variables

CONCEPTOS TEÓRICOS

PARÁMETROS

Son variables que se pasan a una función para que esta pueda trabajar con diferentes valores. Se definen dentro de los paréntesis de la definición de la función.

  • Definición de la Función:

    def saludar(nombre):
    • La palabra clave def se usa para definir una nueva función en Python.
    • saludar es el nombre de la función. Este nombre se usa para identificar y llamar a la función más tarde.
    • nombre es un parámetro de la función. Este parámetro actúa como una variable que acepta un valor cuando se llama a la función.
  • Cuerpo de la Función:
print("¡Hola, " + nombre + "!")
  • Esta línea de código es el cuerpo de la función y se ejecuta cada vez que se llama a la función saludar.
  • print es una función integrada en Python que imprime el mensaje especificado en la pantalla.
  • "¡Hola, " + nombre + "!" es una expresión que concatena la cadena "¡Hola, " con el valor del parámetro nombre y luego con la cadena "!".
  • Si nombre es "Juan", la expresión resultante será "¡Hola, Juan!"

Ejemplo de Uso

  1. Definir la Función: La función saludar se define de la misma manera que antes.

  2. Llamadas a la Función:

    saludar("Juan")
    saludar("María")
    saludar("Pedro")
    • Cada una de estas líneas llama a la función saludar con un argumento diferente.
    • La primera llamada pasa el argumento "Juan", por lo que la función imprimirá "¡Hola, Juan!".
    • La segunda llamada pasa el argumento "María", por lo que la función imprimirá "¡Hola, María!".
    • La tercera llamada pasa el argumento "Pedro", por lo que la función imprimirá "¡Hola, Pedro!".

Interactividad con input

Si deseas que el usuario introduzca el nombre a través de la entrada del teclado, puedes usar la función input:

  1. Pedir Entrada al Usuario:

    nombre_usuario = input("Por favor, introduce tu nombre: ")
    • La función input muestra un mensaje en la pantalla y espera a que el usuario introduzca un valor.
    • El valor introducido por el usuario se almacena en la variable nombre_usuario.

  2. Llamar a la Función con la Entrada del Usuario:

    saludar(nombre_usuario)
    • La función saludar se llama con el valor proporcionado por el usuario.
    • Si el usuario introduce "Ana", la función imprimirá "¡Hola, Ana!".

De esta manera, puedes crear un programa interactivo que saluda al usuario por su nombre.

VARIABLES DE RETORNO

Son los valores que una función devuelve después de ejecutar su código. Se utilizan la palabra clave return.

Paso 1: Definir una función para sumar dos números

def sumar(a, b): return a + b
  • def sumar(a, b): define una función llamada sumar que toma dos argumentos a y b.
  • La función realiza la suma de a y b y devuelve el resultado con return a + b.

Paso 2: Llamar a la función con dos números específicos

resultado = sumar(3, 5)
  • sumar(3, 5) llama a la función sumar pasando los valores 3 y 5 como argumentos.
  • La función sumar calcula 3 + 5 y devuelve 8.
  • El valor devuelto 8 se almacena en la variable resultado.

Paso 3: Imprimir el resultado de la suma

print("La suma es:", resultado)
  • print("La suma es:", resultado) usa la función print para mostrar el mensaje "La suma es:" seguido del valor de la variable resultado.
  • En este caso, resultado contiene el valor 8, por lo que el mensaje completo que se imprimirá es "La suma es: 8".

Funcionamiento

  1. Definir la función:

    • La función sumar toma dos argumentos, a y b, y devuelve su suma.

  2. Llamar a la función con dos números:

    • La función se llama con los números 3 y 5.
    • La función calcula 3 + 5 y devuelve 8.

  3. Imprimir el resultado:

    • El programa imprime el resultado de la suma, que es 8.

Resultado Esperado

Cuando ejecutes el programa, deberías ver algo similar a esto en la pantalla:

La suma es: 8

Consejos

  • Asegúrate de que los números pasados a la función sean del tipo adecuado (en este caso, enteros o flotantes).
  • Usa la función print para mostrar el mensaje de manera clara y legible.
  • Revisa tu código para asegurarte de que no haya errores de sintaxis y que la función esté correctamente definida.

VARIABLES LOCALES

Son las que se definen dentro de una función y solo pueden ser usadas dentro de esa función.

Paso a Paso

  1. Definir la función funcion

    def funcion(): local = 10 # Variable local print(local)
    • def funcion(): define una nueva función llamada funcion.
    • Dentro de esta función, se declara una variable local y se le asigna el valor 10.
    • print(local) imprime el valor de la variable local.

  2. Llamar a la función funcion:

    funcion()
    • funcion() llama a la función funcion, ejecutando el código dentro de ella.
    • La función imprime 10, el valor de la variable local.

  3. Intentar acceder a la variable local fuera de la función:

    print(local) # Esto causará un error porque 'local' no está definida fuera de la función
    • print(local) intenta imprimir el valor de la variable local fuera de la función funcion.
    • Sin embargo, local fue definida dentro de la función funcion, lo que significa que solo existe dentro de esa función.
    • Fuera de la función, local no está definida, lo que produce un error.

Detalle del Error

Cuando ejecutas print(local) fuera de la función, Python no puede encontrar la variable local porque su alcance está limitado a la función funcion. Esto causa un NameError.

Resultado de la Ejecución

Cuando ejecutas el programa, la salida será la siguiente:

  1. La función funcion se llama y se ejecuta:
    10
  2. Después, el programa intenta ejecutar print(local) fuera de la función y produce un error:
    NameError: name 'local' is not defined

Explicación del Alcance de las Variables

  • Variable Local: Una variable definida dentro de una función (como local en este caso) solo está accesible dentro de esa función. Este es el alcance de la variable.
  • Variable Global: Una variable definida fuera de todas las funciones está accesible en cualquier parte del código.

Ejemplo de Variable Local y Global

Para aclarar, veamos un ejemplo con una variable global y una local:

  1. Variable Global:

    • global_variable es una variable global porque está definida fuera de cualquier función.
    • Puedes acceder a global_variable desde cualquier parte del programa.

  2. Variable Local:

    • local es una variable local definida dentro de la función funcion.
    • Solo puedes acceder a local dentro de la función funcion.

Ejecución

  1. La función funcion imprime:
    Dentro de la función: 10
  2. Fuera de la función, se imprime:
    Fuera de la función: 20
  3. Intentar imprimir local fuera de la función causará un error:
    NameError: name 'local' is not defined

Este ejemplo muestra cómo el alcance de las variables afecta dónde pueden ser utilizadas en tu programa. Las variables locales solo existen dentro de su función, mientras que las variables globales existen en todo el programa.

JUEGO CALCULAR LA EDAD EN DÍAS

Explicación del Código

  1. Definición de la Función:

    def calcular_edad_en_dias(edad):
    • def se usa para definir una función en Python.
    • calcular_edad_en_dias es el nombre de la función.
    • edad es un parámetro que recibe la edad en años.

  2. Cálculo dentro de la Función:

    dias = edad * 365 # Aproximación sin considerar años bisiestos
    • dias es una variable que almacena el resultado de multiplicar edad por 365.
    • Este cálculo convierte la edad de años a días, asumiendo que cada año tiene 365 días (sin considerar años bisiestos).

  3. Retorno del Resultado:

    return dias
    • return se usa para devolver el resultado del cálculo desde la función.
    • En este caso, devuelve la cantidad de días.

  4. Entrada del Usuario:

    edad = int(input("Ingresa tu edad en años: "))
    • input("Ingresa tu edad en años: ") muestra un mensaje al usuario y espera a que el usuario introduzca un valor.
    • int convierte la entrada del usuario (que es una cadena) en un número entero.
    • El valor entero se almacena en la variable edad.

  5. Llamada a la Función y Salida del Resultado:

    print("Tu edad en días es aproximadamente:", calcular_edad_en_dias(edad))
    • calcular_edad_en_dias(edad) llama a la función calcular_edad_en_dias con el valor de edad proporcionado por el usuario.
    • print muestra el mensaje "Tu edad en días es aproximadamente:" seguido del resultado de la función.

Ejecución del Programa

Cuando ejecutas este programa, el flujo de ejecución es el siguiente:

  1. El usuario es solicitado a ingresar su edad en años.
  2. El usuario introduce un valor, por ejemplo, 25.
  3. La función calcular_edad_en_dias es llamada con edad igual a 25.
  4. La función calcula 25 * 365, que es 9125.
  5. El resultado 9125 es devuelto por la función.
  6. El programa imprime: "Tu edad en días es aproximadamente: 9125".

Consideración de Años Bisiestos

Este programa no considera años bisiestos. Para una aproximación más precisa, puedes incluir una lógica que cuente los años bisiestos. Aquí hay un ejemplo modificado que incluye esta consideración:

Explicación del Código Modificado

  1. Calcular Años Bisiestos:

    bisiestos = edad // 4
    • bisiestos calcula la cantidad aproximada de años bisiestos en la edad del usuario.

  2. Ajuste del Cálculo Total:

    dias = edad * 365 + bisiestos
    • Este cálculo suma los días adicionales por los años bisiestos.

Este ajuste proporciona una estimación más precisa de la edad en días al incluir los años bisiestos.

ASIGNACIÓN DE TAREAS Y CIERRE

  • Tarea:
    • Escribir una función que reciba dos números y devuelva su suma
  • Cierre:
    • Resumir, en el cuaderno y a mano,  los puntos clave de la sesión

Objetivo

Crea un programa en Python que:

  1. Defina una función para sumar dos números.
  2. Solicite al usuario que ingrese dos números.
  3. Use la función de suma para calcular el resultado.
  4. Imprima el resultado de la suma.

Instrucciones

  1. Define la función para sumar dos números:

    • La función debe tomar dos argumentos y devolver su suma.

  2. Solicita al usuario que ingrese dos números:

    • Usa input para solicitar los números al usuario.
    • Convierte la entrada del usuario a un número de punto flotante (float).

  3. Llama a la función de suma y muestra el resultado:

    • Usa la función print para mostrar el resultado.

Ejemplo de Código

Paso 1: Definir la función para sumar dos números

def sumar(a, b): return a + b
  • def sumar(a, b): define una función llamada sumar que toma dos parámetros, a y b.
  • Dentro de la función, se suma a y b y se devuelve el resultado con return a + b.

Paso 2: Solicitar al usuario que ingrese dos números

numero1 = float(input("Ingresa el primer número: ")) numero2 = float(input("Ingresa el segundo número: "))
  • input("Ingresa el primer número: ") muestra un mensaje al usuario pidiéndole que ingrese el primer número.
  • float(...) convierte la entrada del usuario (que es una cadena de texto) en un número de punto flotante y lo almacena en la variable numero1.
  • Se repite el proceso para el segundo número, almacenándolo en la variable numero2.

Paso 3: Llamar a la función de suma y mostrar el resultado

print("La suma de los dos números es:", sumar(numero1, numero2))
  • sumar(numero1, numero2) llama a la función sumar, pasando numero1 y numero2 como argumentos.
  • La función sumar calcula la suma de numero1 y numero2 y devuelve el resultado.
  • print("La suma de los dos números es:", ...) usa la función print para mostrar el mensaje "La suma de los dos números es:" seguido del resultado de la suma.

Funcionamiento

  1. Definir la función:

    • La función sumar toma dos argumentos, a y b, y devuelve su suma.

  2. Solicitar al usuario que ingrese dos números:

    • El usuario ingresa dos números cuando se le solicita.
    • Las entradas se convierten a números de punto flotante y se almacenan en numero1 y numero2.

  3. Llamar a la función de suma y mostrar el resultado:

    • La función sumar se llama con numero1 y numero2 como argumentos.
    • La función calcula la suma y devuelve el resultado.
    • El resultado se muestra en la pantalla.

Resultado Esperado

Cuando ejecutes el programa, deberías ver algo similar a esto en la pantalla:

Ingresa el primer número: 3.5
Ingresa el segundo número: 4.2
La suma de los dos números es: 7.7

Consejos

  • Asegúrate de ingresar números válidos cuando el programa te lo solicite.
  • Revisa tu código para asegurarte de que no haya errores de sintaxis y que las variables y la función estén correctamente definidas.
