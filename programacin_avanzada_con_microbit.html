1. Acelerómetro

Planteamiento del problema

Queremos crear un programa en la micro:bit que lea la inclinación en el eje X utilizando el acelerómetro. Si la inclinación es mayor a 200, se mostrará ">" en la pantalla LED; si es menor a -200, se mostrará "<"; y si está entre -200 y 200, se mostrará "-". Este proceso debe repetirse continuamente.

Solución con MICROPYTHON

Código en Python para micro:bit:

Explicación del código en Python:

  1. Obtener la inclinación en el eje X: X = input.acceleration(Dimension.X) Lee el valor de la aceleración en el eje X y lo almacena en la variable x.
  2. Condicionales para mostrar símbolos:
    • if x > 200: Muestra ">"
    • elif x < -200: Muestra "<"
    • else: Muestra "-"
  3. Ejecución continua: basic.forever(on_forever) Hace que la micro:bit revise continuamente la inclinación y muestre el símbolo correspondiente.

2. Sensor de temperatura

Planteamiento del problema

Queremos crear un programa en la micro:bit que lea la temperatura ambiente utilizando el sensor integrado y la muestre en la pantalla LED cada segundo. El valor de la temperatura se actualiza continuamente, acompañado de la letra "C" para indicar que la medida está en grados Celsius.

Solución con MICROPYTHON

Código en Python para micro:bit:

Explicación detallada del código en Python:

  1. Obtener la temperatura: temp = input.temperature() Lee la temperatura ambiente en grados Celsius y almacena el valor en la variable temp.
  2. Mostrar la temperatura en la pantalla LED: basic.show_string(str(temp) + "C") Convierte el valor de temp a una cadena de texto, lo concatena con "C", y lo muestra en la pantalla LED.
  3. Pausar antes de la siguiente actualización: basic.pause(1000) Pausa la ejecución del programa durante 1 segundo antes de actualizar la pantalla.
  4. Ejecución continua de la función on_forever: basic.forever(on_forever) Ejecuta continuamente la función on_forever, actualizando la temperatura cada segundo.

    5 Acelerómetro y brújula

    Planteamiento del problema

    Queremos crear un programa en la micro:bit que combine el uso de la brújula y el acelerómetro. El programa debe mostrar la dirección cardinal hacia la que está orientada la micro:bit (Norte, Este, Sur, Oeste) y, al mismo tiempo, indicar si la micro:bit está inclinada hacia adelante, hacia atrás o si está en posición neutral.

    Solución con MICROPYTHON

    Código en Python para micro:bit:

    Explicación detallada del código en MicroPython:

      input.calibrate_compass()

      Esta línea asegura que la brújula se calibre antes de que comience la medición. La calibración es necesaria para que las lecturas de la brújula sean precisas.

angulo = input.compass_heading()

      x = input.acceleration(Dimension.X): Aquí se está midiendo la aceleración de la micro en el eje X, que se refiere a la inclinación hacia adelante o hacia atrás. Si x es positivo, la microestá inclinada hacia adelante, y si es negativo, está inclinada hacia atrás. 

      angulo = input.compass_heading(): Esta línea obtiene la dirección actual de la brújula en grados, con respecto al Norte magnético. El valor se almacena en la variable angulo. Este ángulo varía de 0° a 359°, donde 0° o 360° indica el Norte, 90° el Este, 180° el Sur y 270° el Oeste.

    if angulo < 45 or angulo > 315:

    • Si el ángulo está entre 45° y 135°, la dirección es "E" (Este).
    • Si el ángulo está entre 135° y 225°, la dirección es "S" (Sur).
    • Si el ángulo está entre 225° y 315°, la dirección es "O" (Oeste).

    Dependiendo del valor de angulo, se asigna la dirección cardinal correspondiente a la variable direccion.

    • Mostrar la dirección junto con la inclinación:
    if x > 200:
    basic.show_string(direccion + " >")
elif x < -200:
    basic.show_string(direccion + " <")
else:
    basic.show_string(direccion + " -")

    Este fragmento de código se encarga de mostrar en la pantalla LED de la micro la dirección cardinal junto con un símbolo que indica la inclinación de la micro:bit:

    • Si x es mayor que 200, significa que la micro está inclinada hacia adelante, y se muestra la dirección seguida de " >".
    • Si x es menor que -200, significa que la micro está inclinada hacia atrás, y se muestra la dirección seguida de " <".
    • Si x está entre -200 y 200, significa que la micro está en una posición relativamente neutral, y se muestra la dirección seguida de " -".

    Dependiendo del valor de x, se muestra la dirección cardinal junto con un símbolo que indica si la micro:bit está inclinada hacia adelante (">"), hacia atrás ("<"), o si está en posición neutral ("-").

    • Ejecución continua:
    basic.forever(on_forever)

    Este bloque asegura que el código dentro de la función on_forever se ejecute continuamente, revisando y mostrando la dirección y la inclinación en tiempo real.

      • Añadir Bloque "O" Lógica:
        • Ubicación: Lógica > O.
        • Función: Este bloque combina las dos comparaciones. El operador "O" (OR en inglés) permite que la condición sea verdadera si cualquiera de las dos comparaciones es cierta. Es decir, si el angulo es menor que 45 o mayor que 315, entonces la condición se cumple.

    Conclusión:

    Este programa en MakeCode utiliza una combinación de bloques de Lógica, Entrada, Variables, y Texto para determinar y mostrar la dirección cardinal a la que está orientada la micro, así como para considerar la inclinación de la micro en el eje x. Los bloques de comparación se usan para evaluar el ángulo y la aceleración, mientras que los bloques de unir en Texto se emplean para combinar letras (como "E" u "O") con la dirección cardinal antes de mostrarlas en la pantalla. Cada parte del programa está diseñada para actualizarse continuamente, asegurando que la información mostrada sea precisa y en tiempo real.
